# 五界框架与标签规则 / Five Worlds Framework & Tagging Rules

## 五界概述

根据赫密士哲学的"如在上，如在下"（As above, so below）原则，宇宙存在五个层次，从物质到灵魂依次为：

| 层级 | 英文名 | 中文名 | 说明 |
|------|--------|--------|------|
| I | Physical World | 物质界 | 固态实相，一切可见可触的存在 |
| II | Etheric World | 以太界 | 生命力流动，蓝图与模板的领域 |
| III | Astral World | 星光界 | 情感与欲望，梦境与死后首达之处 |
| IV | Mental World | 心智界 | 纯粹思想与智慧，个体与灵魂的桥梁 |
| V | Spirit World | 灵魂界 | 宇宙意识源头，万物合一的所在 |

---

## 必须标签（归属标签）

每个文件必须包含以下归属标签之一，根据文件内容所在的层级选择：

| 标签 | 层级 | 适用内容 |
|------|------|---------|
| `#I` | 物质界 | 肉体、身体、物质世界、环境、健康、物质层面的一切 |
| `#II` | 以太界 | 生命力、气、脉轮、能量流动、四季轮转、生命力相关 |
| `#III` | 星光界 | 情感、爱恨、恐惧、梦境、欲望、星光界体验 |
| `#IV` | 心智界 | 思想、理念、创造力、智慧、"我存在"意识 |
| `#V` | 灵魂界 | 宇宙意识、万物合一、灵性、回归本源 |

---

## 文件夹结构

项目已建立 I~V 五个文件夹，对应五界：

```
d:\shElter\shElter\
  ├── I\          ← 物质界相关内容
  ├── II\         ← 以太界相关内容
  ├── III\        ← 星光界相关内容
  ├── IV\         ← 心智界相关内容
  └── V\          ← 灵魂界相关内容
```

### 标签与文件夹的对应关系

- 带有 `#I` 标签的文件，建议放置在 `I\` 文件夹中
- 带有 `#II` 标签的文件，建议放置在 `II\` 文件夹中
- 以此类推
- 跨越多个层级的文件，可放置在任意一个相关文件夹中，并在标签中注明所有相关层级

## 自行添加标签

除归属标签外，根据文件内容自行添加描述性标签：

### 常用标签示例

**灵性实践类：**
```
#冥想 #仪式 #灵性练习 #祈祷 #观想
```

**内容类型类：**
```
#每日功课 #记录 #反思 #教程 #概念解释
```

**框架结构类：**
```
#五界 #框架 #体系 #层级
```

**主题类：**
```
#身体 #情感 #思想 #灵性成长 #自我探索
```

---

## 标签放置格式

### 格式一：文件末尾放置

```
正文内容...

---

#I #冥想 #每日功课
```

### 格式二：使用 Frontmatter

```
---
标签:
  - #I
  - #冥想
  - #每日功课
---

正文内容...
```

---

## 标签使用原则

1. **必须有归属标签** — 每个文件至少有一个 `#I` ~ `#V` 的标签
2. **标签数量适度** — 一般 2-5 个标签，不要过多
3. **使用简洁的标签名** — 用单个词或短词组合
4. **统一标签风格** — 全部用中文、或中英混用，保持一致
5. **定期整理标签** — 随着笔记增多，检查是否有重复或过于细碎的标签

---

## 多层级文件

如果一个文件涉及多个界的内容，可以在标签中列出所有相关层级：

```
#II #III #IV     ← 涉及以太界、星光界、心智界
#I #V            ← 涉及物质界和灵魂界
```

## 示例

### 单一层级文件

文件夹：`II\`
文件名：`以太界介绍.md`
标签：

```
---
标签:
  - #II
  - #概念解释
  - #五界
---

## 以太界

内容正文...

---

#II #概念解释 #五界
```

### 多层级文件

文件夹：`I\`
文件名：`从身体到灵魂的旅程.md`
标签：

```
---
标签:
  - #I
  - #II
  - #III
  - #IV
  - #V
  - #框架
---

正文内容讨论从物质界一路上扬至灵魂界的旅程...

---

#I #II #III #IV #V #框架
```
