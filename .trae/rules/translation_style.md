# 翻译与语言风格规则 / Translation & Language Style Rules

## 中英双语交替格式

当使用中英双语交替格式时，必须遵循以下结构：

### 标题格式

```
## [序号]. 【English Title | 中文标题】
```

### 内容格式

每一对中英文解释必须严格遵循以下交替顺序：

```
*[English sentence 1.]*

*[Chinese sentence 1.]*

*[English sentence 2.]*

*[Chinese sentence 2.]*

...
```

### 核心原则

1. **English 先行** — 每组先写英文，再写中文
2. **段落分隔** — 英文和中文之间必须有一个空行
3. **成对呈现** — 每句英文必须有对应的中文翻译
4. **语意完整** — 每一对句子表达同一个完整的意思
5. **不使用连接词合并** — 英文一句、中文一句独立呈现，不使用破折号或冒号连接

---

## 翻译质量原则

翻译不只是转换语言，而是让意思活在新的语言里。遵循以下原则：

1. **自然流畅** — 译文读起来像母语者的自然表达，避免翻译腔
2. **连贯通顺** — 句子之间、段落之间要有自然的承接和过渡
3. **常态化用词** — 用日常、亲切、易懂的词汇取代学术黑话
4. **意思不变** — 在让中文更通顺的前提下，严格保留英文原意
5. **避免直译陷阱** — 不要逐字硬翻，优先译意而非译字

### 常见翻译改进

| 不推荐的译法 | 推荐译法 | 原因 |
|-------------|---------|------|
| 固态实相 | 坚实、可触的真实 | "实相"过于抽象 |
| 物质实相 | 坚实、可触的真实 | 同上 |
| 能量 | 生命力 / 能量（根据语境） | "能量"是黑话 |
| 永恒的智慧 | 超越时间的智慧 | "永恒"比"超越时间"更难理解 |
| 一切都 | 所有...都 | "一切"是文言遗风 |
| 首先抵达的领域 | 首先到达的地方 | "领域"过于抽象 |
| 更加轻盈 | 更轻盈 | "更加"冗余 |
| 完整成形 | 成形 | "完整"冗余 |

---

## 示例

正确格式：

```
## I. 【Physical World | 物质界】

*The lowest world, solid reality.*

*最底层的世界，坚实、可触的真实。*

*All visible and tangible beings manifest here.*

*一切可见可触的存在，都在此处显化为真实。*
```

错误格式（翻译腔/不流畅）：

```
## I. 物质界 / Physical World

*最底层的世界，固态实相。* *The lowest world, solid reality.*
```

或：

```
## I. 【Physical World | 物质界】

*The lowest world, solid reality.* — *最底层的世界，固态实相。*
```
