# switch 语句语法要求

## 1. 表达式类型要求

*switch 后面的表达式必须是==整型或枚举类型==，不能是 float 或 double 类型。*

*表达式类型限制确保了比较的准确性和效率，浮点数由于精度问题不适合用于分支判断。*

## 2. case 标签要求

*case 后面必须是一个==整型常量表达式==，不能是浮点数或变量（除非是 const 整型常量）。*

*常量表达式确保了编译时即可确定分支目标，提高执行效率。*

## 3. 标签顺序与唯一性

*switch 语句体中 case 和 default 标签可任意顺序出现，且 case 标签值必须唯一。*

*标签值唯一避免了歧义，顺序灵活但建议按逻辑顺序排列。*

## 4. 语法结构要求

*switch(...) 后面不能直接加分号，否则后面的大括号就会独立成一个块，switch 没有实际控制。*

*正确的语法结构确保了 switch 语句能够正确控制流程。*
